# LLC-Flow MVP

LLC-Flow is an AI-driven platform designed to simplify the formation of Limited Liability Companies (LLCs) in the United States. It guides users through a conversational interface powered by Claude AI to collect necessary data and generates professional, state-specific Articles of Organization.

## Features
- **Conversational AI Agent**: Powered by Claude 3.5 Sonnet to handle complex legal data collection.
- **State-Specific Document Templates**: Currently supports Delaware and California.
- **Dynamic PDF Generation**: Generates official filing documents in real-time.
- **JSON Data Export**: Allows users to download their structured data for easy reference.

## Project Structure
- `/client`: React + Vite frontend with Tailwind CSS and Framer Motion.
- `/server`: Node.js + Express backend with Anthropic Claude SDK and jsPDF.
- `/shared`: Shared configurations and state logic.

## Getting Started

### Prerequisites
- NPM or Yarn
- Anthropic Claude API Key

## Environment Variables
The backend requires a `.env` file in the `/server` directory:
```env
CLAUDE_API_KEY=your_api_key_here
CLAUDE_MODEL=claude-3-5-sonnet-20240620
```

### Easy Start (Unified)
This is the fastest way to run both the frontend and backend together:
1. Make sure you have installed dependencies in both `/client` and `/server`.
2. Run the start script from the root:
   ```bash
   ./start.sh
   ```
   *Press `Ctrl+C` to stop both servers at once.*

### Manual Start (Individual)
If you prefer to run the services in separate terminals:

#### Backend Setup (Server)
1. Navigate to the server directory:
   ```bash
   cd server
   ```
2. Create a `.env` file and add your Claude API Key (see `.env.example`).
3. Start the development server:
   ```bash
   npm run dev
   ```
   The server will run at `http://localhost:3001`.

#### Frontend Setup (Client)
1. Navigate to the client directory:
   ```bash
   cd client
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
   The application will be accessible at `http://localhost:5173`.

## How it Works
1. **Selection**: Choose your formation state on the landing page.
2. **Chat**: Interact with the AI agent to provide business names, addresses, and member details.
3. **Verify**: Review the extracted data for accuracy.
4. **Download**: Download your professional Articles of Organization PDF and instructions for manual state filing.

## Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Lucide, Framer Motion.
- **Backend**: Express, TypeScript, Anthropic API, jsPDF.
- **Tools**: GitHub CLI for deployment, Git for version control.
